Detailed Description

Implementation of calculator that uses recursive parser.

The calculator parses mathematical expressions using next grammar: 

G = E '\0'
E = T ([+|-] T)*
T = F ([*|/] F)*
F = P (^ P)*
P = '(' E ')' | N
N = [0-9]+
